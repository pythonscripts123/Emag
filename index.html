Sites That Allow Game Embedding

  • itch.io (some games) — HTML5 Games
    Many developers enable embedding. Look for "Embed" option on the game page.
  • GameMonetize — gamemonetize.com
    Hundreds of embeddable HTML5 games. Each game has an iframe embed code.
  • Gamedistribution — gamedistribution.com
    Register for access. Many games are provided with embed codes.
  • OpenGameArt (HTML5 Games) — OpenGameArt HTML5 Games
    Open-source HTML5 games and demos, often embeddable and modifiable.
  • Construct Arcade — construct.net
    Many games are embeddable. Always check the game's page for embed or share options.
  • Some open-source HTML5 games on GitHub
    You can self-host and embed these games if the license allows.

Note: Always check the terms and use the provided embed code. Some platforms (like CrazyGames, Poki, and Miniclip) do not allow embedding.
